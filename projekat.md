# Predikcija performansi računarskog hardvera korištenjem regresijskih modela



U okviru projekta, koristeci Computer Hardware Dataset koji sadrži tehničke karakteristike računarskih sistema (npr. brzina procesora, memorija), trenirana su dva modela masinskog ucenja sa ciljem predikcije performansi. Koristen je python sa modelima iz biblioteka scikit-learn. Ridge Regression model se pokazao boljim od Random Forest modela.
